# GearUp Backend API 
Sports & Outdoor Gear Rental Platform Backend.

## Tech Stack
- Node.js & Express.js
- TypeScript
- PostgreSQL (Neon DB)
- Prisma ORM
- Zod Validation
- Stripe Payments

## Local Setup
1. Clone the repository.
2. Run `npm install`.
3. Configure your `.env` variables.
4. Run `npx prisma db push` followed by `npm run seed`.
5. Run `npm run dev`.
